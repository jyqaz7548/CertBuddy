# CertBuddy 새로운 개발 계획

## 개발 철학

**프론트엔드 우선 개발 (Frontend-First Development)**

1. 프론트엔드를 먼저 완성하여 UI/UX를 확정
2. 프론트엔드 요구사항에 맞춰 백엔드 API 설계
3. Mock 데이터로 모든 기능 동작 확인
4. 백엔드 구현 후 통합
