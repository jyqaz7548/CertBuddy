### Phase 1 완료 후 → Phase 2: 데이터베이스 설계

#### 우선순위 1: 핵심 테이블 설계
1. **users** 테이블
   - 사용자 기본 정보
   - 학년, 학과 정보
   - XP, 스트릭 정보

2. **certifications** 테이블
   - 자격증 정보
   - 학교/학과별 추천 정보

3. **flash_cards** 테이블
   - 플래시카드 내용
   - 자격증별 카드 그룹

#### 우선순위 2: 학습 관련 테이블
4. **learning_sessions** 테이블
   - 학습 세션 기록
   - XP 획득 기록

5. **review_cards** 테이블
   - 복습 필요한 카드 저장

## MVP vs 확장 기능 구분

### MVP (최우선 개발)
1. ✅ 로그인/회원가입
2. ✅ 홈 화면 (기본 대시보드)
3. ✅ 일일 학습 (플래시카드)
4. ✅ 복습 모드
5. ✅ 마이페이지

### 확장 기능 (MVP 이후)
1. 🔄 친구 기능 (친구 추가, 랭킹)
2. 🔄 함께 학습하기
3. 🔄 선배 취업 자격증 리스트
4. 🔄 시험 일정 조회
5. 🔄 학교/학과 기반 추천 (고도화)
